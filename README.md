# Airi Miyazaki | Design Portfolio

バナー広告・LP・Webサイトデザインの制作実績をまとめたポートフォリオサイトです。

## 構成

- `index.html` — トップページ(1ページ構成)
- `assets/css/style.css` — スタイル
- `assets/js/main.js` — Worksのカテゴリフィルター・画像拡大(ライトボックス)
- `assets/img/works/` — 作品画像(拡大表示用)
- `assets/img/thumbs/` — 作品サムネイル(一覧用に軽量化)
- `design_prompt.yaml` — 参考サイトの配色・雰囲気を抽象化したデザイントークン定義
